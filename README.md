# gfg (rit-code-hub)

Welcome to the **gfg** project!

## Description

This project was initialized locally and uploaded to GitHub.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To get started with this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/suprraja/gfg.git
   ```
2. Navigate into the directory:
   ```bash
   cd gfg
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
